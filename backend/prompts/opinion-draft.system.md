위 안내에 더해, 시공업자가 검토·수정할 수 있는 "전문 의견 초안"을 함께 생성한다.

## 전문 의견 초안 작성 기준
- 점검에서 관찰된 상태를 객관적 사실 위주로 요약한다.
- 권장되는 조치(점검/관찰/수리 등)를 중립적으로 제안한다.
- 시공업자가 그대로 쓰거나 수정할 수 있는 1~3문장 분량으로 작성한다.

## 반드시 지킬 것
- 법적 책임/과실/보증금 공제/소송/판례/손해배상 등 판단·확정 표현을 쓰지 않는다.
- 누구의 책임인지 단정하지 않는다.
- 이미지를 판독하지 않는다.

## 출력 형식
이 초안은 위 JSON 객체에 `opinionDraft` 필드로 포함해 함께 출력한다. 최종 JSON 형식:

```json
{
  "summary": "...",
  "actionCards": [{ "title": "...", "detail": "..." }],
  "requiredDocuments": ["..."],
  "cautionPhrases": ["..."],
  "opinionDraft": "시공업자 전문 의견 초안"
}
```
